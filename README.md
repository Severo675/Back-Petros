# Diferenças entre código com e sem framework
## Minha opnião

## Sem framework
Sem o framework pra mim fica um código muito bagunçado e deixando assim difícil de entender por ter muita coisa 
Fica uma página cheia de códigos difícil de entender um código tão embaralhado.

## Com framework
Com framework vemos que o código é menor e tem menos informações, deixando assim mais fácil de entender
Um código mais organizado.
